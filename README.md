# 🎯 Pomodoro Focus Timer

A clean, minimal Pomodoro-style focus timer — runs entirely in the browser, no installs needed.

**[→ Live Demo](https://aditya282.github.io/Pomodoro-Focus-Timer/)**

---

## What it does

- Full Pomodoro cycle — 4 focus sessions, then a long break
- Color-coded ring: gold for focus, blue for short break, purple for long break
- Session dots track where you are in the cycle
- Audio alert when a session ends
- Adjustable durations for all three phases
- Works on any device — desktop, tablet, phone

## The Pomodoro Technique

| Session | Default Duration |
|---|---|
| 🟡 Focus | 25 min |
| 🔵 Short Break | 5 min |
| 🟣 Long Break | 15 min (after 4 focus sessions) |

## Built with

- Vanilla HTML, CSS, JavaScript — zero dependencies
- Web Audio API for sound
- SVG ring animation for the countdown
